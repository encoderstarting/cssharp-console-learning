# C# Console Learning

Учебный консольный проект на C#.

## О проекте
Приложение реализовано в виде меню и содержит несколько функций:
- Проверка оценки
- Проверка возраста
- Калькулятор

Проект создан для практики основ C# и работы с Git.

## Как запустить
Открыть файл `ConsoleApp4.sln` в Visual Studio и запустить проект (F5).

Или через терминал:
```bash
dotnet run
